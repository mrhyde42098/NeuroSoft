# NeuroSoft Beta - Guia rapida para Mayra

## Instalacion

1. Ejecute `NeuroSoft_Beta_Setup.exe`.
2. Elija la carpeta de instalacion cuando el asistente lo pregunte.
3. Deje marcado el acceso directo del escritorio si lo quiere.
4. Abra NeuroSoft desde el acceso directo o desde el menu Inicio.

Los datos se guardan localmente en `%APPDATA%\NeuroSoft`. Al desinstalar la app, esa carpeta no se borra automaticamente para evitar perdida accidental de informacion clinica.

## Credenciales de prueba

- Administrador: `admin` / `neurosoft2025`
- Beta tester: `mayra` / `MayraBeta2026!`

El usuario `mayra` puede probar los flujos clinicos, pero no tiene permisos para backups, auditoria, KPIs ni configuracion administrativa.

## Que reportar

Reporte cualquier error con este formato:

```text
TIPO: Bug / Usabilidad / Interpretacion clinica
MODULO: Pacientes / Evaluacion / Screening / Informes / Rehab / Otro
PASOS: Que hizo antes de ver el problema
RESULTADO: Que paso
ESPERADO: Que esperaba que pasara
IMPACTO: Bajo / Medio / Alto / Critico
CAPTURA: Adjuntar screenshot si aplica
```

## Nota para envio por Gmail

Gmail suele bloquear archivos `.exe` incluso dentro de `.zip`. La forma mas confiable es subir `NeuroSoft_Beta_Setup.exe` a Google Drive, OneDrive o WeTransfer y enviar el enlace.
